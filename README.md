# 📊 Customer Data Hub Dashboard

## 🚀 Overview

This project is a Customer Data Hub Dashboard built using Python and Streamlit.
It helps users upload, analyze, and visualize customer data from multiple sources in a simple and interactive way.

The main goal of this project is to make data analysis easier and faster by combining multiple datasets into one platform and generating useful insights.

---

## 🧠 Features

* 📂 Supports multiple file formats (CSV, JSON, Excel)
* 📊 Interactive data visualization using charts
* ⚡ Real-time data processing
* 🧹 Automatic data cleaning and handling
* 🤖 AI-based insights generation (optional feature)
* 📄 Data extraction from structured files

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Altair
* Google Generative AI (for insights)

## 📊 How It Works

1. Upload customer datasets
2. Data is processed using Pandas
3. Visualizations are generated
4. AI provides insights (if enabled)
5. Results are shown in dashboard
